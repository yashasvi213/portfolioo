# Portfolio Landing (React + TypeScript + Vite)

## Stack
- React 18.3.1 + TypeScript
- Vite 5.4.19
- TailwindCSS 3.4.17
- React Flow + D3 + Recharts
- Radix UI primitives (shadcn-style `ui` components)
- Framer Motion
- React Hook Form + Zod
- TanStack React Query
- React Router DOM 6.30.1
- Lucide React icons
- Prettier + tailwind plugin

## Folder structure

```text
src/
  components/
    layout/
    sections/
    ui/
  pages/
  lib/
```

## Run

```bash
npm install
npm run dev
```
